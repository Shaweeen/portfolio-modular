# Modular Portfolio Site Documentation

## Introduction
This documentation covers how to use and customize the Modular Portfolio Site.

## Getting Started
1. **Clone the Repository**  
   To get started, clone the repository using:  
   ```bash  
   git clone https://github.com/Shaweeen/portfolio-modular.git  
   ```  

2. **Install Dependencies**  
   Navigate to the project directory and install the necessary dependencies:  
   ```bash  
   cd portfolio-modular  
   npm install  
   ```  

## Usage
- Start the local development server:  
  ```bash  
  npm start  
  ```  
- Open your browser and navigate to `http://localhost:3000` to view the site.

## Customization
1. **Modify the Content**  
   Update the content in the `src` directory to reflect your portfolio items.  

2. **Change Styles**  
   The styles can be changed in the `src/styles` directory. You can modify the CSS files to fit your design needs.

3. **Add New Sections**  
   To add new sections, create new components in the `src/components` folder and import them into the main application file.
